Assessment

Overall Result
The model/provider does not provide a trustworthy translation-quality confidence value by default.

Usable signals
1. Native response metadata has no semantic confidence field.
2. `logprobs` can be requested and may be useful for token-level diagnostics.
3. `logprobs` should not be shown as translation correctness confidence.
4. Prompt-requested `confidence` is model self-report and can be confidently wrong.

Recommendation
Do not add a user-visible confidence score based only on model self-report or token logprobs.

If the product needs reliability signaling, prefer labels such as:
1. "needs context"
2. "literal translation may be misleading"
3. "expression note available"
4. "unverified expression"

These should be derived from explicit probe-validated behavior, dictionary/glossary support, or separate verification passes, not from self-reported model confidence alone.
